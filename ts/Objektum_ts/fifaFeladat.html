  1. Először létrehozunk egy string típusú tömböt csapatAdat néven, amelyben objektumokat tárolunk. Ezek az objektumok a csapatok adatait tartalmazzák.

    "Anglia;4;0;1662",
    "Argentína;10;0;1614",
    "Belgium;1;0;1752",
    "Brazília;3;-1;1719",

  2. Ezután definiálunk egy interfészt fifaElem néven, amely meghatározza az objektumok struktúráját. Az interfészben négy tulajdonság szerepel: csapat, helyezes, valtozas és pontszam. Ezek a tulajdonságok megadják a csapat nevét, helyezését, változását és pontszámát.

    interface fifaElem {
    csapat: string;
    helyezes: number;
    valtozas: number;
    pontszam: number;
    }

    3. A ObjektumFeltolto nevű függvényben megvalósítjuk az objektumok létrehozását a feltöltendő elemek alapján. A függvény paramétere egy string tömb (feltoltendoElem), amely a csapatok adatait tartalmazza.

      function ObjektumFeltolto(feltoltendoElem: string[]): fifaElem[] { var beolvasottAdatok: fifaElem[] = [];
      for (let i: number = 0; i < feltoltendoElem.length; i++) {
      let darabolandoSor: string[] = feltoltendoElem[i].split(";");
      let objektum: fifaElem = {
      csapat: darabolandoSor[0],
      helyezes: Number(darabolandoSor[1]),
      valtozas: Number(darabolandoSor[2]),
      pontszam: Number(darabolandoSor[3]),
      } beolvasottAdatok.push(objektum);
      } return beolvasottAdatok;
      } var fifaAdatok = ObjektumFeltolto(csapatAdat);

    1. Létrehozunk egy üres tömböt, beolvasottAdatok néven, amely a létrehozott objektumokat fogja tárolni.
    2. Végigmegyünk a feltoltendoElem tömb elemein egy for ciklus segítségével. Minden elemet felosztunk pontosvessző (;) mentén, és létrehozunk egy új objektumot a darabolandoSorban található adatokkal.
    3. Az objektum tulajdonságait a darabolandoSorban található értékekkel töltjük fel. A csapat tulajdonságot a darabolandoSor első elemével, a helyezes tulajdonságot a második elemével, a valtozas tulajdonságot a harmadik elemével, míg a pontszam tulajdonságot a negyedik elemével állítjuk be. A Number() függvény segítségével a szöveges értékeket számmá konvertáljuk.
    4. Az elkészült objektumot hozzáadjuk a beolvasottAdatok tömbhöz a push() metódussal.
    5. Végül visszaadjuk a beolvasottAdatok tömböt, amely az objektumokat tartalmazza.
    4. Az AdatKiiro nevű függvényben megjelenítjük az objektumok adatait. A függvény paramétere egy fifaElem tömb (adatok), amely az objektumokat tartalmazza.

      function AdatKiiro(adatok: fifaElem[]): void {
      var tabla = document.createElement('table');
      // Táblázat fejléce
      var tablaOsztaly = 'tablaOsztaly'; //táblaOsztaly létrehozása
      tabla.classList.add(tablaOsztaly); //hogy lehessen css-ben különböző formázásokat végrehajtani
      // Táblázat css tulajdonságok módosítása

      var tablaFejlec = document.createElement('tr');//
      var csapatFejlec = document.createElement('th');
      csapatFejlec.textContent = 'Csapat';
      var helyezesFejlec = document.createElement('th');
      helyezesFejlec.textContent = 'Helyezés';
      var valtozasFejlec = document.createElement('th');
      valtozasFejlec.textContent = 'Változás';
      var pontszamFejlec = document.createElement('th');
      pontszamFejlec.textContent = 'Pontszám';
      tablaFejlec.appendChild(csapatFejlec);
      tablaFejlec.appendChild(helyezesFejlec);
      tablaFejlec.appendChild(valtozasFejlec);
      tablaFejlec.appendChild(pontszamFejlec);
      tabla.appendChild(tablaFejlec);
      // Táblázat sorai
      for (let i: number = 0; i < adatok.length; i++) {
      var sor = document.createElement('tr');
      var csapatCella = document.createElement('td');
      csapatCella.textContent = adatok[i].csapat;
      var helyezesCella = document.createElement('td');
      helyezesCella.textContent = adatok[i].helyezes.toString();
      var valtozasCella = document.createElement('td');
      valtozasCella.textContent = adatok[i].valtozas.toString();
      var pontszamCella = document.createElement('td');
      pontszamCella.textContent = adatok[i].pontszam.toString();
      sor.appendChild(csapatCella);
      sor.appendChild(helyezesCella);
      sor.appendChild(valtozasCella);
      sor.appendChild(pontszamCella);
      tabla.appendChild(sor);
      }
      document.body.appendChild(tabla);
      }
      AdatKiiro(fifaAdatok);

    • Először létrehozunk egy tabla nevű table elemet.
    • Dom művelet érdekében létrehozzuk a tablaOsztaly nevű változót, és ezt az osztályt hozzáadjuk a táblázathoz
    • Létrehozunk egy táblázat fejlécet (tablaFejlec) és hozzáadjuk a táblázathoz. A fejléc tartalmazza a "Csapat", "Helyezés", "Változás" és "Pontszám" címeket.
    • A for ciklus segítségével végigmegyünk az adatok tömbön, és minden objektumhoz létrehozunk egy új sort (sor) a táblázatban.
    • Az objektum tulajdonságait tartalmazó cellákat (csapatCella, helyezesCella, valtozasCella, pontszamCella) létrehozzuk, és beállítjuk a megfelelő értékeket az objektum tulajdonságainak megfelelően.
    • A cellákat hozzáadjuk a sorhoz.
    • A sort (sor) hozzáadjuk a táblázathoz (tabla).
    • Végül a táblázatot (tabla) hozzáadjuk a dokumentum testéhez (document.body).
    • Az AdatKiiro függvény meghívásával megjelenítjük az objektumok adatait a táblázatban.
